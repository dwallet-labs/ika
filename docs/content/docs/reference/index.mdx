search and skimming, not read top to bottom.

  module paths and the inner BCS types.
- [Network configs](./network-configs): package IDs, object IDs, and
  RPC endpoints for testnet, mainnet, and how to assemble one for
  localnet.
- [Move modules](./move-modules): packages and modules deployed by
