This section is for the operators of Ika rather than the applications
that use it.

- **[CLI](./cli)**: the `ika` command-line tool. dWallet commands,
  validator commands, config commands.
- **[Validator setup](./validator-setup)**: hardware, keys, and
  initial configuration to join the validator set.
- **[Validator operations](./validator-operations)**: monitoring,
  upgrades, key rotation, incident response.
- **[Networks](./networks)**: testnet, mainnet, and localnet
  endpoints, plus how to run a local stack for development.

If you are an application developer, this section is mostly
optional. The exception is [Networks](./networks), which covers
localnet for hands-on development and which mainnet/testnet endpoints
to trust.
